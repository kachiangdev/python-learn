üìö What are Dictionaries?

Dictionaries are like real-world dictionaries, but instead of words and definitions, they store key-value pairs. Think of them as a way to organize related information - like a phone book where names (keys) are linked to phone numbers (values).

üóùÔ∏è Dictionary Features

  • Store data as key-value pairs for easy lookup
  • Keys must be unique, but values can be duplicated
  • Very fast for finding specific data
  • Mutable - you can add, remove, or change entries
  • Perfect for representing real-world relationships

Dictionary Operations

  • dict[key] - Access value by key
  • dict[key] = value - Add or update key-value pair
  • dict.get(key) - Safely access value (returns None if key doesn't exist)
  • dict.keys() - Get all keys
  • dict.values() - Get all values
  • dict.items() - Get all key-value pairs
