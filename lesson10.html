üîÑ What are Loops?

Loops are one of the most powerful features in programming! They let you repeat tasks automatically instead of writing the same code over and over. Whether you're processing a list of items or counting to 100, loops make it easy.

üéØ Types of Loops

  • for loops - Perfect for going through lists or counting
  • while loops - Keep going until a condition changes
  • Range loops - Repeat a specific number of times
  • List loops - Process each item in a collection
  • Dictionary loops - Iterate through key-value pairs
  • Nested loops - Loops inside other loops

Loop Types

  • for item in list: - Loop through list items
  • for i in range(n): - Loop n times (0 to n-1)
  • for i in range(start, end): - Loop from start to end-1
  • for key in dict: - Loop through dictionary keys
  • for key, value in dict.items(): - Loop through key-value pairs
  • while condition: - Loop while condition is True

Exercise 2: Sum Calculator
