üì• What is User Input?

User input allows your programs to be interactive! Instead of having fixed values, you can ask users to provide information that your program can use. This makes your programs dynamic and personalized.

üí° Key Facts About Input

  • The input() function always returns a string
  • You can display a prompt message to guide users
  • Convert to numbers using int() or float()
  • Always consider what type of data you expect
  • Input makes programs interactive and engaging

Example 3: Mathematical Operations

Key Points

  • input() always returns a string
  • Use int() to convert to integer
  • Use float() to convert to decimal
  • The prompt text is optional but helpful
  • Always handle potential conversion errors
