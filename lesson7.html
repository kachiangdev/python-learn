Lesson 7: 🎭 Python Adventure Generator

🎯 Review all concepts from lessons 1-6 with a fun, interactive project!

🌟 Welcome to Your First Big Project!

Congratulations! You've learned the fundamentals of Python programming. Now it's time to put everything together in one exciting project: the Python Adventure Generator!

🎮 What You'll Build

A fun, simple program that:

  • 👤 Gets your name and details
  • 🧮 Calculates character stats using simple math
  • 🎯 Lets you choose your adventure location
  • 📜 Creates a personalized story with your choices
  • 🏆 Gives you a rating based on your stats
  • 🎉 Uses every concept from lessons 1-6!

📚 Concepts We'll Review

📝 Lesson 1: Printing

Format adventure stories and character info

📦 Lesson 2: Variables & Types

Store character stats, names, and adventure data

🧮 Lesson 3: Arithmetic

Calculate character power, treasure values

🔤 Lesson 4: String Magic

Create dynamic stories with f-strings

📥 Lesson 5: User Input

Get player choices and preferences

🤔 Lesson 6: Conditions

Make adventure decisions and branching stories

🎯 Project: Simple Adventure Story

Let's build a fun story generator step by step! Each step focuses on one concept and can be run by itself.

🏁 Step 1: Welcome the Player (Lesson 1: Printing)

# Simple welcome message print("🎭 WELCOME TO THE ADVENTURE STORY MAKER! 🎭") print("Let's create your adventure!") print()

👤 Step 2: Get Player Information (Lesson 5: Input)

# Get player name name = input("What's your name? ") print("Hello, " + name + "!") print()

🎂 Step 3: Get More Details (Lesson 5: Input + Lesson 2: Variables)

# Get more player info age = int(input("How old are you? ")) favorite_color = input("What's your favorite color? ") # Store the information print("Got it! You are " + name + " and you are " + str(age) + " years old.") print("Your favorite color is " + favorite_color + ".") print()

🧮 Step 4: Calculate Character Points (Lesson 3: Arithmetic)

# Simple math calculations strength = age + 10 magic = len(name) + 5 luck = len(favorite_color) + 3 # Show the results print("=== YOUR CHARACTER STATS ===") print("Strength: " + str(strength)) print("Magic: " + str(magic)) print("Luck: " + str(luck)) print()

🎯 Step 5: Choose Your Adventure (Lesson 6: Conditions)

# Simple choice print("Where do you want to go?") print("1. Forest") print("2. Castle") print("3. Cave") choice = input("Pick 1, 2, or 3: ") # Simple if-else if choice == "1": location = "forest" monster = "wolf" elif choice == "2": location = "castle" monster = "ghost" else: location = "cave" monster = "dragon" print("You chose the " + location + "!") print("You meet a " + monster + "!") print()

📜 Step 6: Create Your Story (Lesson 4: String Formatting)

# Create a simple story using f-strings print("=== YOUR ADVENTURE STORY ===") print(f"Once upon a time, {name} went to a {location}.") print(f"With {strength} strength and {magic} magic power,") print(f"{name} bravely faced the {monster}!") print(f"Using {favorite_color} armor, our hero won the battle!") print()

🏆 Step 7: Give a Rating (Lesson 6: Simple Conditions)

# Simple rating system total_points = strength + magic + luck print("=== ADVENTURE RATING ===") print("Total Points: " + str(total_points)) # Simple rating if total_points > 30: rating = "AMAZING HERO" elif total_points > 20: rating = "GOOD ADVENTURER" else: rating = "BRAVE BEGINNER" print("You are a " + rating + "!") print("Thanks for playing!")

🎯 Your Mission: Build the Complete Story!

🚀 Main Challenge: Complete Adventure Story

Combine all the code sections above into one complete program! Copy each step in order and run it to create your personalized adventure story.

💡 Pro Tips:
  • Copy each step in order (1, 2, 3, 4, 5, 6, 7)
  • Run the program multiple times with different inputs
  • Try different names, ages, and choices to see how the story changes
  • Notice how every concept from lessons 1-6 is being used!

🎨 Bonus Challenges
